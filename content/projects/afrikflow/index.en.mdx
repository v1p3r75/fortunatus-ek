AfrikFlow is a revolutionary web and mobile application designed to facilitate financial transactions in West Africa. This project focuses specifically on the UEMOA (West African Economic and Monetary Union) zone and uses local payment aggregators as well as major mobile money operators to offer a simple, fast and secure money transfer service.

### Key Features

- **Fast and Secure Money Transfers**: Send and receive money instantly thanks to our integration with operators such as Moov Money, Mobile Money, Orange CI, Celtiis Cash, and many others.
- **Multi-Platform**: Available in both web and mobile versions, offering maximum accessibility for all users.
- **Integration with Local Aggregators**: Use local payment aggregators for better coverage and smooth transactions.
- **Intuitive User Interface**: A user-friendly and easy-to-use interface for an optimal user experience.
- **Enhanced Security**: Implementation of best security practices to protect your transactions and personal data.
- **Multilingual Support**: Available in multiple languages ​​to serve a diverse audience within the UEMOA zone.
